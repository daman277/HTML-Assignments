HTML, CSS, JavaScript
Front-end technologies used to create the user interface and interactivity of a web application.
React
A JavaScript library for building interactive user interfaces, often used for creating dynamic front-end components.
Node.js
A runtime environment that allows JavaScript to run on the server side, enabling server-side scripting for web applications.
Express.js
A minimal and flexible Node.js web application framework used to build robust and scalable server-side applications.
MongoDB
A NoSQL database that stores data in JSON-like documents, commonly used for handling data in web applications.
SQL (e.g., MySQL, PostgreSQL)
Relational databases used for structured data storage, retrieval, and management in web applications.
RESTful APIs
A set of architectural constraints for designing networked applications, allowing communication between the front-end and back-end.
Git
A distributed version control system for tracking changes in code, enabling collaboration and code management.
Webpack
A module bundler that helps manage and optimize the assets and dependencies of a web application.
Heroku
A cloud platform that simplifies deployment and hosting of web applications, making them accessible on the web.
