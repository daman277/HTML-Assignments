HTML Tags

  1. !DOCTYPE: Defines the document type and version of HTML being used. It helps the browser understand how to interpret the HTML content.
  2. html: The root element of an HTML page. All other elements are nested within this one.
  3. head: Contains meta-information about the document, such as the title of the page and links to external resources like stylesheets and scripts.
  4. title: Sets the title of the HTML document, which appears in the browser's title bar or tab.
  5. meta: Provides metadata about the document, such as character encoding and author information.
  6. link: Used to link external resources like stylesheets to the HTML document.
  7. style: Defines internal CSS styles for the document.
  8. script: Specifies client-side JavaScript code or links to an external JavaScript file.
  9. body: Contains the visible content of the web page, including text, images, links, and more.
  10. header: Represents a container for introductory content or a set of navigational links, often found at the top of a page.
  11. nav: Defines a section of navigation links.
  12. main: Contains the main content of the page, excluding headers, footers, and sidebars.
  13. article: Represents a self-contained composition within a document, such as a blog post or news article.
  14. section: Groups related content together, typically with a heading.
  15. aside: Defines content that is tangentially related to the content around it, often used for sidebars.
  16. footer: Contains the footer information for a section or the entire page.
  17. p: Represents a paragraph of text.
  18. a: Creates hyperlinks to other web pages or resources.
  19. img: Embeds images on the page.
  20. ul: Defines an unordered (bulleted) list.
  21. ol: Defines an ordered (numbered) list.
  22. li: Represents a list item within ul or ol elements.
  23. div: A generic container for grouping and styling content, often used with CSS
  24. span: A generic inline container for applying styles or scripting to a specific section of text.
  25. br: Inserts a line break within text.
  26. hr: Adds a horizontal line or thematic break.
  27. table: Defines a table.
  28. tr: Represents a table row.
  29. th: Defines a table header cell.
  30. td: Represents a table data cell.
